# Student Performance & Engagement Analytics Dashboard

Tools: Power BI | DAX | Power Query | Data Modeling

## Data Source :  Education Analysis Link


## Project Summary

Developed an end-to-end interactive analytics dashboard to evaluate student performance, engagement, assessment outcomes, and registration trends within an online learning environment. The project transforms raw academic data into actionable insights to support data-driven academic decision-making.


---

## What I Built

Designed 6 fully interactive Power BI dashboards covering:

Executive Overview

Student Profile

Student Performance

Engagement (VLE)

Assessment Details

Registration Analysis


Created optimized data models and relationships across multiple fact and dimension tables

Implemented DAX measures for KPIs such as pass rate, average scores, retention rate, engagement metrics, and studied credits

Applied dynamic filters and slicers (module, region, gender, education level, assessment type)



---

## Key Insights Delivered

Identified module-wise performance gaps and pass rate variations

Analyzed student engagement behavior and its impact on academic outcomes

Tracked assessment submission trends and score distributions

Highlighted withdrawal and retention patterns across demographics and modules

Compared performance by education level, gender, and region

## Business / Academic Value
